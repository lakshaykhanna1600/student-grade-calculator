# 🎓 Student Grade Calculator

A CLI-based Python project that calculates student grades based on subject-wise marks.

## 🚀 Description

This program allows users to input multiple subjects along with their maximum marks and obtained marks. It calculates the percentage for each subject, assigns grades (A–F), and displays detailed results.

At the end, it also computes the overall percentage, final grade, and pass/fail status.

## ✨ Features

* Input any number of subjects
* Calculates percentage for each subject
* Assigns grades (A, B, C, D, F)
* Displays overall percentage and final grade
* Shows pass/fail result

## 🛠️ Technologies Used

* Python (CLI)

## 📚 Concepts Practiced

* Loops (`for`)
* Conditional statements (`if-elif-else`)
* Functions
* User input handling

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/lakshaykhanna1600/student-grade-calculator.git
   ```

2. Navigate to the project folder:

   ```bash
   cd student-grade-calculator
   ```

3. Run the program:

   ```bash
   python main.py
   ```

## 📌 Example Flow

* Enter number of subjects
* Enter subject name, max marks, and marks obtained
* Get subject-wise percentage and grade
* View overall result at the end
