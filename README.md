# 🎓 Student Grade Calculator

A CLI-based Python project that calculates student grades based on subject-wise marks.

## 🚀 Description

This program allows users to input multiple subjects along with their scores (out of 100). It assigns a grade for each subject, and at the end displays the overall percentage and final grade.

## ✨ Features

- Input any number of subjects
- Assigns grades (A, B, C, D, F) per subject
- Displays overall percentage and final grade

## 🛠️ Technologies Used

- Python (CLI)

## 📚 Concepts Practiced

- Loops (`for`)
- Conditional statements (`if-elif-else`)
- Functions
- Lists
- User input handling

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/lakshaykhanna1600/student-grade-calculator.git
```

2. Navigate to the project folder:
```bash
cd student-grade-calculator
```

3. Run the program:
```bash
python main.py
```

## 📌 Example Flow

- Enter number of subjects
- Enter subject name and score (e.g. `Math:98`)
- Get subject-wise grade
- View overall percentage and final grade at the end
